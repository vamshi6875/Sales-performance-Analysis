# Sales-performance-Analysis

## Project Objective
This project analyzes sales data and uncover business insights. It mainly concentrates on overall sales, profit margin and regional performance.
## Dataset used
- <a href="https://github.com/vamshi6875/Sales-performance-Analysis/blob/main/SuperStoreUS-2015%20(1).xlsx">Dataset</a>
## Tools used
- Microsoft Excel
- Power BI
## Process
veried data for any missing values and removed duplicates.
Data transformation done for fixing errors and inconsistent data.
Used KPI's and Slicers for exploring performance.
Created dashboard to visualize the data in Power BI.

## Project Insights
- South is the only region facing losses of approximatly 14k.
- Regular Air is the only shipping mode that 90% of the people use.
- Highest number of quantities are sold from Office Supply.
- West region is the only place where delivery truck dominated regular air 

  ## Final Conclusion
  To recover the loss in Technology from South Region the best stratergy is to combine the products category.
  To boost up the sales of office supplies a good amount of discounts & offers should be applied than regular one.
